***---Dans ton terminal, place‑toi dans le dossier du projet et tape :***



&nbsp;		chmod +x git-save.sh





***---Lancer le script avec ton message de commit À chaque sauvegarde, tu exécutes :***



&nbsp;		./git-save.sh "Mon message de commit"





***---Changer l’URL du dépôt si tu passes sur un autre projet Si tu veux lier ton script à un nouveau repo GitHub :***



&nbsp;		git remote set-url origin https://github.com/tonpseudo/nouveau-projet.git





***Tu n’as pas besoin de modifier le script, juste l’URL du dépôt distant.***
